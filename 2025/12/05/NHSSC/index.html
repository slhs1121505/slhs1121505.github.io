技藝競賽心得

張子夫2025-12-052025-12-05

前言

阿，終於比完技藝競賽了，好累好累。
我整個十一月都在選手訓練，回來感覺要補一堆功課（悲

很高興自己能獲得第十名，原本就沒期望說可以拿到前三，能有 25% 的技優加分我就很爽了哈哈。
但不過我應該接下來會繼續考 APCS 跟 CPE，資安方面則是考個 iPAS 然後比個 AIS3 Pre-exam，英文方面則是多益拚個藍色。

第一天

就是我爸開車從我家載到士商，結果我到的時候，零個人在穿堂…？
等了二十幾分鐘才陸續有幾個人過來，然後我們科的老師都在最後的時候才上車，嗯，可能比較塞車。

其實第一天也沒啥事，就只是過去報到，然後因為我們學校是觀摩團（下屆要辦），所以就有一個很大的教室，哈哈爽。
聽說如果不是的話就要擠在下面，我最後一天頒獎典禮有去坐，超擠。

然後就搞一搞設備檢測完就去飯店入住了，學校真的很有錢欸，給我們住四星級的飯店，我跟會計的選手住，好像三晚要一萬多耶。


喔還有，誰懂飯店裡面有免治馬桶的救贖感，看到免治馬桶就感覺像是在礦洞裡面挖到鑽石的感覺。

然後我就跟我指導老師去吃摩斯漢堡，因為我怕衛生的問題，比賽完之前都只吃連鎖哈。
結果我點到有起司的漢堡，我不喜歡起司，但也只能硬著頭皮吃，哭。

吃完就回去練下術科了，我 LCS 還不會。（不過最後是沒考 la）

第二天

五點多起來就洗漱下就去吃飯店早餐了。
第一天我還記得我吃的時候，由於太緊張，我手抖到我想用叉子叉香腸都叉不下去。

然後就比術科～～

我其實有點忘記細節了，也就那樣，有一題平方數的，由於他測資有 5000 筆，然後數字為 (0 < n < 10^9)。
我一開始是想說用 Python 內建的 pow() 跟 int 去判斷是否為平方數，但是因為測資太大，這樣一個一個搞就會 TLE。

然後我就突然想到我上次考 CPE 的第四題，考嚴格遞增字典序，是可以用 itertools 去建 List 然後去做查詢。
然後我就試試看建一個 0 到 10^9 都是 False 的 List，然後跑迴圈去把 i*i 的都變成 True，然後 i+1 這樣，然後就過了，耶。

我最後成績是 16 題，第一第二名都是 19 題，太鬼了。
我沒過的那三題是質因數分解、表格 DP、最小生成樹。

我當下看到最小生成樹天都塌了你知道嗎。
然後質因數那個用 Sieve of Eratosthenes 會 TLE，後面回去問 ChatGPT 才知道說是要用試除法。
然後表格 DP 當下是真沒想到推導式，不過大概能知道是三個條件這樣。

總之 16 題我就很爽了，因為可以知道自己穩前十五名。

由於考完術科是大概十一點多，吃完飯就回去飯店了。
其他學校的選手好像還要擠在小小的禮堂..？ 哈

然後這邊不得不說，我們學校訂的飯店真的好美，愛士商。

下午就沒啥事，就一直在休息，然後我室友回來，我們兩個就去湯姆熊打彈珠，好玩。

晚餐我想說還是繼續吃衛生一點的，路邊攤我怕真的會拉。
然後我就找到一家有賣家常菜的，就自己過去吃。


點完才發現有點貴，哈哈，這樣要 295，但還好學校晚餐都給我們 250 當餐費，愛士商！

然後吃完回去就要讀學科，不過我那時候真的不知道要讀啥，就背了下運算子優先順序這樣。
其他時間我都在看電視這樣，然後室友感覺很認真，都在讀會計。

第三天

依舊吃飯店早餐，我覺得飯店的早餐就那幾個而已，所以我都吃香腸跟蛋而已。

然後就比學科～～

考前我才想到好像要帶手錶，但是回去拿不知道能不能來得及，就不拿了，打算就直接去比。
阿不過幸好裡面有時鐘，耶。

然後我前面有一個神人，好像是 C026，第一天設備檢測不會用 VS Code 被監考老師念，第三天比學科，進去之前問老師裡面有沒有附 2B 鉛筆。
我是覺得 Bro 你要不要直接回家算了。

阿現場都沒有人要借，呵呵。

今年學科我覺得，應該算簡單，沒有太大的迴圈這樣。
但有考幾個冷門函數，像 enumerate、all、any 啥的。

enumerate 還好我以前題目有寫過，但我自己是比較喜歡隨便創一個變數當計數器啦。
然後另外兩個就是看字面意思。

喔然後有考一個 -3 % 4 的，我當下超緊張，想說靠邀完了，我不會負數的。
阿不過我就猜 3 跟 4 差 1，所以就猜 1，結果對了。

出來就直接回去觀摩團的休息室對學科答案，結果滿分，超爽的。
我之前寫歷屆都沒有滿分，哈哈哈有夠爽。

對完我就在南商隨便走走，認真覺得南商好大，我們學校好小，明年我們學校真的能辦得成嗎..？


然後就中午回去觀摩室甲奔，吃完就回飯店休息了。
其他選手跟老師好像是旅行社帶他們去其他地方玩。

回飯店前有買手搖，叫特好喝，是丁特開的。
不得不說台南的三分糖還是好甜，然後特好喝我怕喝多了以後小孩會被墮胎。

啊我就在飯店廢廢廢廢這樣。



兩點多時候，老師就買豆花給我吃，愛老師。
聽說這個豆花是很有名的，但我吃不出來哪裡有名這樣。

然後由於飯店的馬桶真的真的很爛，所以我都拿飯店送的攪拌棒拿來通馬桶，耶。

所以當我要用飯店附的紅茶、砂糖跟奶精搞奶茶的時候，就沒有攪拌棒可以用。
所以我就想到可以拿兩個杯子搞印度拉茶。

晚上我原本是要吃旭集，但是因為太貴，加上我都沒有吃到台南的特色美食，我就沒有去吃了。

我跟我老師去國華街逛，一路上聊了很多升學的，然後我買了特選牛肉湯，有附牛肉燥飯，爽。
還有買了香酥雞、薯條跟花枝丸。

我跟室友原本晚上是要等入圍名單出來，結果好像是因為會計的比賽有問題，就延到隔天五點五十九分才出來。

聽說是比會計的時候有人穿學校的運動服，被站在樓下的老師看到，好像是某義某商的老師。
阿老師就想搞死那個穿運動服的，就向主辦單位舉報這樣（不過是我也會想要搞死其他選手 la）。
然後主辦單位就要調監視器啥的，就拖拖拖。

第四天

阿最後我是有進入圍，雖然我一開始就知道了，十六題要不進也是很難。

然後就在飯店又又又吃了香腸跟蛋。

最後就頒獎～～

我能有第十名，我真的真的就已經，很爽了。
應該是可以上個雲科資管吧，應該，求求雲科收收窩。


然後就回去了，睡睡睡睡睡。

阿我爸要去看醫生不能來載我，所以我就拖著行李箱搭公車搭捷運搭公車這樣，命苦。

洗完澡就躺床上了，醒來才發現，幹，我絕區零沒打式與防衛戰。

結語

嗯，真的很高興可以獲得第十名，也希望以後能靠這個成績上個好大學這樣。

其實我一路學來，我越來越沒有信心，程式設計有太多太多的東西是我需要去學習的。

這時候鼓勵我的一句話，是 Girls Band Cry 裡面，主角井芹仁菜很常穿的衣服上面的話：

「Hope for the best and prepare for the worst.」

因為這句話，我才有了信心，使我有動力繼續學習。

總之，真的很感謝很感謝，一路上所有幫助過我的人，真的，非常非常，感謝。
